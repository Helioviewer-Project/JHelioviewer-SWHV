News

July 1, 2011: JHelioviewer 2.1.4 is out

Due to the growing user community of JHelioviewer and helioviewer.org, our server traffic has increased significantly. This bug fix release, together with the deployment of our new open-source JPIP server addresses a number of bugs, some of which were related to high server load.

Feb 24, 2011: JHelioviewer 2.1.3 is out

This update adds support for SDO/HMI data and features a new contrast filter, as well as an improved plugin to access the Heliophysics Event Knowledgebase (HEK).

Dec 16, 2010: Amazing Feedback

This week, the new JHelioviewer release was presented at the AGU Fall Meeting in San Francisco. Starting off with two articles on the ESA Portal and Science & Technology web sites, the JHelioviewer story started multiplying and was soon covered by dozens of articles around the globe. Within two days, the JHelioviewer videos on YouTube have been watched more than 400,000 times. Thanks to everyone for this amazing feedback!

Dec 13, 2010: JHelioviewer 2.1 is out!

The new JHelioviewer 2.1 is now available. Apart from all new features of November's release candidate (2.1 RC), this version contains a new RGB channel mixer as well as many bug fixes.

Nov 11, 2010: JHelioviewer 2.1 RC released

We are happy to announce the new JHelioviewer release candidate! JHelioviewer 2.1 RC provides many new features:

  • Access to SDO/AIA data
  • Versatile movie export
  • Plugin to access the Heliophysics Event Knowledge Base
  • Platform-specific bundles for Mac OS X and Windows

Check out the Helioviewer Project Blog for more news and up-to-date information.
