JHelioviewer Tutorial

This tutorial walks you through the basic steps of using JHelioviewer to explore solar image data. Make sure you have downloaded and installed JHelioviewer before starting.

Step 1: Launch JHelioviewer

Double-click the JHelioviewer icon to start the application. After a few seconds the main window appears, showing the most recent solar images.

Step 2: Load a Data Layer

Click File → New Image Layer to open the data browser. Select an observatory, instrument, and detector from the drop-down menus, then choose a date and time range and click OK.

Step 3: Play a Movie

Use the playback controls at the bottom of the window to step through the image time series. Press the Play button (›) to start the animation, and the Stop button to pause it.
